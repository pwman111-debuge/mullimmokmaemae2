# 아이디어 노트

## 눌림목매매2 × 포트폴리오분석 콜라보 구상 (2026-04-26)

### 핵심 인사이트

두 프로젝트는 겹치지 않고 완벽하게 상호보완적이다:

- **포트폴리오분석** = "어떤 종목을 살 것인가" (글로벌 수급 + 섹터 매크로 + 펀더멘털)
- **눌림목매매2** = "언제 살 것인가" (눌림목 건강도 점수 + 타점 확인)

### 제안 구조

```
새 프로젝트 "마스터에이전트" (가칭)
         ↓
1단계: 포트폴리오분석 → 이번 주 후보 종목 3~5개 추출
         ↓
2단계: 눌림목매매2 → 해당 종목들의 건강도 점수 + 타점 확인
         ↓
3단계: 교집합 → "펀더멘털 OK + 타이밍 OK" 종목만 진입 권고
         ↓
4단계: 포트폴리오분석 대장에 반영
```

### 방법론 비교 요약

| 항목 | 포트폴리오분석 | 눌림목매매2 |
|------|-------------|-----------|
| 핵심 질문 | 어떤 종목을 살 것인가 | 언제 살 것인가 |
| 종목 선정 | 글로벌 수급 + 펀더멘털 | 과거 리포트 추이 분석 |

| 건강도 스코어링 | ❌ 없음 | ✅ 핵심 |
